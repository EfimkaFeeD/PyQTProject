Пожалуйста, правильно заполняйте информацию, обращайте внимание на все знаки и заглавные буквы.
Старайтесь по минимум использовать пробелы в названиях файлов.
Названия жанров и типов пишите с маленькой буквы. Есть следующие типы: фильм, сериал, короткометражное (все что короче 60 минут).
Файлы с описанием и актерами загружать в папку text, актеров в casts, описания в descriptions только в формате txt. При добавлении не добавлять .txt, достаточно только названия.
Файлы с изображениями и постерами добавляются в папаку images, изображения в film_images, постеры в posters.
Актеров в файл добавлять по шаблону: Имя ' ' Фамилия ' ' Имя на английском ' ' Фамилия_на_английском ':' Роль которую играет на языке оригинала
В сборы добавлять только число в долларах, без знаков.
В рейтинге на Rotten Tomatoes добавлять оценку аудитории.
